Запуск происходит с EXE файла далее для работы ничего не нужно.
В первом окне выбираем под каким пользователем хотим работать(Сотрудник или Бухгалтер).
Далее в зависимости от выбора открывается соответствующее окно
В окне сотрудник мы можем задать ФИО, выбрать вид справки, выбрать количество копий, указать причину запроса справки.
При нажатии кнопки "Отправить заявку" данные передаются в файл если файла нет и приложение было только что запущенно файл создаётся новый, даные записываются в новый файл.
Далее для просмотра справок необходимо указать ФИО все остальные поля могут быть пустыми нажимаем кнопку "Просмотреть справки" открывается дополнительное окно со всеми справками на данное имя(Не реализовано исключение совподения по ФИО разных пользователей)
При нажатии кнопки "Назад" возвращаешься на стартовое окно.
В окне для бухгалтера первым делом сразу считывается файл со всеми справками и выводится на экран.
Имеются поля для выбора справки в которой будет вносится изменения статуса, и поле статуса который будет установлен.
При нажатии кнопки "Применить" При верном выборе статуса произойдет смена(Не реализованно отслеживание уже пройденных этапов статуса.)
При нажатии кнопки "Назад" возвращаешься на стартовое окно.
При нажатии кнопки "Закрыть приложение" Происходит соответствующее событие.
